 Manual

Start Video Report

Copy the game number from Basketplan. Make sure that the game has the video-icon set, otherwise it will not work, since no YouTube video is associated with this game.

Paste the game number into the input-field and press "Search".

Select the referee you want to review and press "Start"

Creating the Report

At any time you can save the current state of the report. If you return to the homepage, you can edit it again by pressing the edit-icon.

Creating Video Comments

The corresponding YouTube video is displayed. Start the video. Once you found the beginning of a scene you want to comment on, pause the video and press the button "Create comment for current timestamp"). At the right side you can now add the comment for this scene.

Finishing the Report

Once you confirm to finish the report, the report will become read-only and the reviewed referee will receive an email containing the link to this report (same as pressing on the icon next to the finish-icon).
